## Specify Format

Request specific content types:

```
You: create a comparison brief for "bamboohr vs gusto"
```

```
You: create a how-to brief for "setting up employee onboarding"
```

```
You: create a listicle brief for "10 hr tools every startup needs"
```

Claude adjusts the outline structure based on content type.

---

## Include Competitor Angle

When your brand has competitors defined, briefs automatically include positioning:

```
DIFFERENTIATION ANGLE:
- Emphasize simplicity vs BambooHR's complexity
- Highlight pricing transparency vs Gusto
- Focus on startup-specific features
```

## Brief Components

| Section | Purpose |
|---------|---------|
| Target keyword | Primary SEO focus |
| Intent | Informational, commercial, transactional |
| Word count | Recommended length |
| Outline | H1-H3 structure |
| Featured snippet | Position 0 target |
| Internal links | Cross-linking opportunities |

---

## Tips

**Be specific with keywords.** "hr software" is vague. "hr software for startups under 50 employees" gives better briefs.

**Include intent.** "comparison brief" vs "how-to brief" changes the structure significantly.

**Check brand context.** Run `/brand` first. Briefs use your competitors and positioning.
